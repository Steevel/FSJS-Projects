# 🔖 Bookmark App 📑

This is a bookmark app developed using HTML, TailwindCSS and Javascript.

## 🔗 Deployed Link

[Live Link](https://bookmarkmanagerapp.netlify.app/)

## ✨ Features

- Add bookmark
- Remove bookmark
- Visit bookmarked site
- Beautiful UI
