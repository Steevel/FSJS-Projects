# 🎨 Colour Picker App 🌈

This is a colour picker app developed using HTML, TailwindCSS and Javascript.

## 🔗 Deployed Link

[Live Link](https://color-select.netlify.app/)

## ✨ Features

- Beautiful UI
- Rich colors to choose from
- Click to copy
