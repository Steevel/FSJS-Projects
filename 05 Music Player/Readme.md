# 🎼 Music Player 🎧

This is a music player app developed using HTML, TailwindCSS and Javascript.

## 🔗 Deployed Link

[Live Link](https://play-songs.netlify.app/ "Live Link")

## ✨ Features

- Play/pause music
- Volume up/down
- Previous/next track
- Seekbar
- Repeat track
